# 自定义端口说明

## 方法 1：通过环境变量临时修改（推荐）

在启动开发服务器时指定端口：

```bash
# 使用自定义端口 3000
PORT=3000 pnpm dev

# 使用自定义端口 8080
PORT=8080 coze dev

# 或使用 npm/pnpm 命令
PORT=3000 pnpm run dev
```

## 方法 2：永久修改默认端口

编辑 `scripts/dev.sh` 文件，修改默认端口：

```bash
PORT=3000  # 将 5000 改为你想要的端口
```

## 方法 3：使用 .env 文件

创建 `.env.local` 文件：

```bash
PORT=3000
```

## 支持的端口范围

- 推荐端口：3000-9000
- 禁止端口：9000（系统保留）
- 常用端口：3000、4000、5000、8000

## 注意事项

⚠️ **重要：**
- 端口 9000 被系统保留，禁止使用
- 确保端口未被其他服务占用
- 如果端口被占用，脚本会自动尝试释放
- 部署环境必须使用 **5000** 端口（不可修改）

## 示例

```bash
# 使用端口 3000
PORT=3000 pnpm dev

# 使用端口 8080
PORT=8080 bash ./scripts/dev.sh
```
