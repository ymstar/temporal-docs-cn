## 术语对照表

| 英文术语 | 中文翻译 | 说明 |
|---------|---------|------|
| Workflow | 工作流 | 定义业务流程的确定性代码 |
| Activity | 活动 | 执行非确定性操作的代码 |
| Temporal Server | Temporal 服务器 | 中央协调器 |
| Worker | Worker | 运行工作流和活动的进程 |
| Signal | 信号 | 向运行中的工作流发送数据 |
| Query | 查询 | 查询工作流状态 |
| Update | 更新 | 同步更新工作流状态 |
| Task Queue | 任务队列 | Worker 订阅的队列 |
| Execution | 执行 | 工作流的一次运行实例 |
| RunID | 运行ID | 工作流执行的唯一标识符 |
| Workflow ID | 工作流ID | 工作流的业务标识符 |
| Namespace | 命名空间 | 工作流的隔离空间 |
| Event History | 事件历史 | 工作流执行的完整历史记录 |
| Determinism | 确定性 | 相同输入产生相同输出的特性 |
| Retry Policy | 重试策略 | 活动失败时的重试规则 |
| Child Workflow | 子工作流 | 由父工作流启动的工作流 |
